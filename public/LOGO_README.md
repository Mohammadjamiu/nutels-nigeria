# NUTELS Logo

Please add the NUTELS logo file to this directory (`public/`) with the filename `nutels-logo.png`.

The logo should be:
- A circular design with a blue gear/cogwheel
- White outline
- Contains a map of Nigeria (green and white) with a telecommunications tower
- Text: "NATIONAL UNION OF TELECOMMUNICATIONS STUDENTS" and "NIGERIA"

The logo is referenced in:
- `app/components/Navigation.tsx` (header navigation)
- `app/page.tsx` (homepage hero section)

Once added, the logo will automatically appear in these locations.
